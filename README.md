# 📿 Ultimate Digital Tasbeeh Counter (ڈیجیٹل تسبیح کاؤنٹر)

An ultra-premium, modern, and highly interactive **Digital Tasbeeh Counter** web application designed to help Muslims around the world keep track of their daily Azkar, supplications, and remembrance of Allah. Built using clean, native web technologies (HTML5, CSS3, ES6+ JavaScript), this app is fully responsive, lightweight, and offline-capable (Progressive Web App ready).

---

## 🌟 Key Features (نمایاں خصوصیات)

*   **🌍 Multilingual Support (تین زبانیں):** Easily switch interface languages between **Arabic (العربية)**, **Urdu (اردو)**, and **English** with proper Right-to-Left (RTL) and Left-to-Right (LTR) layouts.
*   **🌓 Elegant Themes (لائٹ اور ڈارک موڈ):** Premium dark mode designed for comfortable nightly recitation, along with an elegant light mode.
*   **📖 Authentic Azkar Database (مستند احادیث سے ثابت شدہ اذکار):**
    *   **Morning & Evening Supplications (صبح و شام کے اذکار)**
    *   **Post-Prayer Remembrance (نماز کے بعد کے اذکار)**
    *   **General Tasbeeh (عام تسبیحات)**
    *   *Every Zikr features accurate Arabic text, translation (Urdu/English), and reliable Sahih Hadith references.*
*   **🔊 Immersive Multi-Sensory Feedback:**
    *   **Click Sound:** Clean, high-pitched synthetic click tone (powered by Web Audio API).
    *   **Haptic Vibration Feedback:** Vibrates your mobile device upon reaching completion targets.
*   **📊 Smart Target Tracking:** Set a custom target count (e.g., 33, 100). The beautiful SVG circular progress bar fills dynamically as you tap, accompanied by a celebration audio-visual alert upon completion.
*   **📱 PWA Ready:** Built with a `manifest.json` config for native installation on home screens of Android, iOS, and desktop devices.

## 🚀 Quick Setup & Installation (انسٹالیشن اور طریقہ کار)

As a completely client-side application, you don't need to run any heavy backend servers.

### Method 1: Local Run (اپنے کمپیوٹر پر چلائیں)
1. Clone this repository or download the ZIP file to your local computer:
   ```bash
   git clone https://github.com/yourusername/digital-tasbeeh-counter.git
   ```
2. Navigate into the project folder.
3. Simply double-click `index.html` to open it in any modern web browser (Chrome, Edge, Safari, Firefox).

### Method 2: Deployment (انٹرنیٹ پر لائیو کریں)
You can deploy this static application completely for free on platforms like:
*   **GitHub Pages:** Push your code to a GitHub repository, go to **Settings** > **Pages**, and choose the `main` branch.
*   **Vercel / Netlify:** Drag and drop this project directory into their dashboard for instant automated deployment.

## 🛠️ Built With (ٹیکنالوجی)

*   **HTML5** - Structured and accessible semantic markup.
*   **CSS3** - Custom CSS Variables for real-time theme swapping, smooth transitional animations, and responsive grids.
*   **Vanilla JS (ES6+)** - Reactive event handling, state control, and **Web Audio API** (synthesizes audio without external assets).
*   **Google Fonts** - *Amiri* (for elegant Arabic text), *Noto Nastaliq Urdu*, and *Poppins* (for clear UI labels).
*   **FontAwesome Icons** - Modern, vector-based interactive icons.
